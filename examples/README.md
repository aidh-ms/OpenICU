# OpenICU examples

This directory separates the main OpenICU pipeline examples from adapter/export examples.

- `openicu_pipeline/`: reserved for end-to-end OpenICU extraction/concept/sharding examples.
- `ricu_converter/`: examples for generating OpenICU YAML concept configs from RICU `concept-dict.json`.
- `yaib_export/`: examples for exporting OpenICU concept outputs to YAIB/RICU-like dynamic wide tables.

The adapter examples use small YAML config files so paths and run parameters can be changed without editing notebook code.
